TikTok API OAuth Demo Flow

1. 앱 시작 화면

사용자는 http://localhost:5000 에 접속하여 TikTok 로그인을 시작합니다.

화면: "Login with TikTok" 버튼

2. TikTok OAuth 권한 요청

TikTok 로그인 창이 나타나며 사용자 권한 요청을 승인합니다.

예시 URL: https://www.tiktok.com/v2/auth/authorize?...

요청 권한: 사용자 기본 정보, 업로드한 영상 목록

3. 인증 성공 후 콜백

OAuth 인증이 완료되면, 앱은 /callback 경로로 리디렉션되며 Access Token을 획득합니다.

예시 Access Token: eyJhbGciOi...xyz

4. API 호출 - /video/list

발급받은 Access Token으로 TikTok API 호출 결과:
